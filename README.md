# First-Purs

First PureScript Code i.e. Hello World

## Development Guide

#### NOTE : Please make sure you have spago :: [Installing spago](https://github.com/spacchetti/spago#installation)

* Initial setup

```bash
$ spago install
```

* Build

```bash
$ spago build
```

* Run

```bash
$ spago run
```

* To Run and watch for changes

```bash
$ spago run --watch
```
